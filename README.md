# Apple Battery Health Analyzer

Analyze Apple battery plist payloads to extract cycle count, battery health,
charging state, and related diagnostics. Paste or upload a plist file, then
review the parsed metrics in a clean, static UI.

## Features

- Parses battery plist XML from iOS or macOS diagnostics.
- Extracts cycle count, health, capacities, voltage, and temperature.
- Flags health bands and charging state.
- Runs entirely in the browser with no data upload.

## Getting Started

```bash
npm install
npm run dev
```
